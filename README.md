# Lycaeides Genomic Offset
Code for *Lycaeides* genomic offset project.

## Directories

**Annotation:** Gene and functional annotations.

**Climate:** Estimation of current, past, and future climates.

**Demography:** Regression of demographic parameters on genomic offset.

**Enrichment:** Enrichment tests for genes and ontologies.

**GEA:** Genotype-environment-association analysis.

**Temporal:** Temporal genomic offset analysis.
